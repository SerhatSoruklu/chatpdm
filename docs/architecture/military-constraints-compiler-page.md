# Military Constraints Compiler Page

The public page at `/military-constraints-compiler` is a static inspection surface for the Military Constraints Compiler.

## What It Is

- A bounded, public explanation page.
- A read-only summary of the compiler surface.
- A place to show the decision contract, the locked pack surface, and a static example payload.
- A page that uses a hero image and editorial copy, not a live decision console.

## What It Is Not

- Not a tactical decision tool.
- Not a military advice surface.
- Not a backend client.
- Not a freeform scenario interpreter.
- Not a live evaluation console.

## Current Sections

- Hero image and title
- What it is
- What it is not
- Decision contract
- Pack overview for Packs 1 through 5
- Static example JSON

## Current Behavior

- The page is static.
- The page does not call the backend.
- The page does not accept user input for evaluation.
- The page does not expose reviewed clauses.

## Public Framing

The page describes the subsystem as a protocol-grade admissibility engine:

- deterministic
- refusal-first
- versioned
- traceable
- bounded to validated packs
It does not present the subsystem as a strategy engine, recommendation engine, or targeting surface.
